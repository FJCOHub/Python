# Portafolio: Análisis de Ventas

Este proyecto realiza un **Análisis Exploratorio de Datos** sobre un dataset de ventas por tienda y mes.

## Dataset
- Contiene las ventas de 2 tiendas durante 8 meses.
- Archivo: `data/ventas.csv`
